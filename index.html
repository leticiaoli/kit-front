Aprendendo a usar o Gulp

Um kit de ferramentas para automatizar e aprimorar seu fluxo de trabalho

Aproveite o gulp e a flexibilidade do JavaScript para automatizar fluxos de trabalho lentos e repetitivos e compô-los em pipelines de construção eficientes.

Conectando plugins

Usar plug-ins criados pela comunidade é uma maneira rápida de começar a usar o gulp. Cada plug-in faz uma pequena quantidade de trabalho, para que você possa conectá-los como blocos de construção. Encadear plugins de várias tecnologias para alcançar o resultado desejado.

Navegue pelos plugins da comunidade para ver o que está disponível!
